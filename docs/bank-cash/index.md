# Bank / Cash Module

The Bank/Cash module manages all bank and cash-related operations in QBS ERP, from master data setup to payment, receipt, and transfer transactions.

## Sub-Modules

| Sub-Module | Purpose |
|------------|---------|
| [Bank](bank) | Define and manage bank institutions |
| [Bank Branch](bank-branch) | Manage branches for each bank |
| [Bank Account](bank-account) | Register and manage company bank accounts |
| [Bank Transaction Type](bank-transaction-type) | Define transaction categories for bank entries |
| [Payment](payment) | Manage payment batches and transactions |
| [Receive](receive) | Manage receipt batches and transactions |
| [Transfer](transfer) | Manage inter-bank transfer transactions |

---

{: .note }
The Bank, Branch, and Transaction Type modules must be configured before creating Bank Accounts and recording transactions.
