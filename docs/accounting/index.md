# Accounting Module

The Accounting module is the financial backbone of QBS ERP. It provides tools to manage your General Ledger, Chart of Accounts, Vouchers, and Financial Year settings.

## Sub-Modules

| Sub-Module | Purpose |
|------------|---------|
| [General Ledger](general-ledger) | Manage ledger batches and post voucher entries |
| [Chart of Accounts](chart-of-accounts) | Manage individual GL accounts |
| [Chart of Accounts Type](chart-of-accounts-type) | Define account categories (Asset, Liability, etc.) |
| [Voucher Type](voucher-type) | Define transaction types (Payment, Receipt, etc.) |
| [Financial Year](financial-year) | Manage accounting periods |

---

{: .note }
Only users with the **Administrators** role or equivalent accounting permissions can access setup modules.
