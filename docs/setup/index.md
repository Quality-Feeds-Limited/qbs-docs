# Setup Module

The Setup module contains configuration data for geographic locations and business classifications used across QBS ERP.

## Sub-Modules

| Sub-Module | Purpose |
|------------|---------|
| [Division](division) | Top-level geographic classification |
| [District](district) | Districts within each division |
| [Area](area) | Areas within each district |
| [Business Type](business-type) | Client and outlet business categories |
